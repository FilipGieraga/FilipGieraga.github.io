API Project

This project allows you to find books from NY Times API. There are more than 50 types of book categories.
By default the date is set to current and the API can run without any input in that field.

I tested different endpoints for many categories and it seems like the API always finds information for current date, but sometimes struggles to find books when the date is provided. If something goes wrong, the message is displayed to try again.

By clicking on a book you can explore more details and there is a link to buy it on Amazon.
This is my first Bootstrap 5 project.
